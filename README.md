# Responsive Webpage - CODTECH Internship

## 📌 Project Description
This project is a responsive webpage built using HTML, CSS, and JavaScript.  
It adapts to different screen sizes such as mobile, tablet, and desktop.

## 🚀 Features
- Responsive design using CSS media queries
- Navigation bar with mobile toggle menu
- Clean and simple UI
- Works on all devices

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript

## 📱 Responsiveness
The webpage adjusts layout based on screen size using media queries.

## 🎯 Outcome
A fully functional responsive webpage demonstrating adaptability across devices.
